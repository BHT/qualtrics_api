# QualtricsAPI

Ruby wrapper for Qualtrics REST ControlPanel API version 3.0.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'qualtrics_api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install qualtrics_api
